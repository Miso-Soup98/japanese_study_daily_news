# 每日新闻日语学习系统

双击 `生成今日新闻.bat`，系统会：

1. 从 Google 新闻日文版聚合的多家媒体中获取当天候选；
2. 为政治、经济、娱乐、体育、社会、数码科技、军事分别评分并选择一条；
3. 从七个专业栏目中再选影响最高的一条作为“新闻”头条；
4. 使用本机离线模型翻译为中文；
5. 使用日语词法分析为汉字添加振假名；
6. 在 `output` 文件夹中生成 HTML 报告并自动打开。

## 交互式学习解析

- 在日文标题或正文中用鼠标选中单词、短语或句子，右侧会显示词形、读音、原形、词性、活用和语法候选。
- 词典数据来自 EDRDG 的 JMdict/EDICT。生成程序最多每 30 天更新一次本地缓存，网页只嵌入当天新闻实际使用到的词条。
- “高可信”只用于词形、读音完全匹配且候选义项较少的词；多义词自动降为“中可信”，并列出候选义项，不自动替学习者选择语境义。
- JMdict 未找到完全一致词条时标为“低可信”，页面不会自动编造确定释义。
- 中文释义若来自自动翻译，会明确标注为辅助内容；权威词典义项保留英文原文。
- 语法结果属于基于人工整理规则的形式匹配，并显示正文依据。容易产生歧义的形式会标为“中匹配”。

JMdict/EDICT 数据由 Electronic Dictionary Research and Development Group
维护，并按其词典许可使用：

- https://www.edrdg.org/jmdict/j_jmdict.html
- https://www.edrdg.org/edrdg/licence.html

评分考虑来源权威度、发布时间、重大事件关键词和栏目相关性。自动筛选不能替代编辑判断，
因此每条内容均保留来源链接，建议阅读原报道确认细节。

系统生成的是学习用途摘要，而不是新闻正文复制。

## 自动更新与网站发布

- GitHub Actions 按日本时间每天 00:00 自动运行。
- 最新日报同时写入仓库根目录的 `index.html`，可直接作为 Cloudflare Pages 的静态网站入口。
- 历史日报保存在 `output` 文件夹中。
- 页面顶部提供八个栏目按钮，可直接跳到对应新闻。
